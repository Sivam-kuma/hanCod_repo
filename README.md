# HanCod Service App

## Features
- Firebase Authentication (Phone & Google)
- Supabase Backend
- Cleaning Services Listing
- Cart Management
- Riverpod State Management

## Tech Stack
- Flutter
- Riverpod
- Firebase Auth
- Supabase
- GoRouter

## How to Run
1. Clone repo
2. Add Firebase config
3. Add Supabase keys
4. Run `flutter pub get`
5. Run app
